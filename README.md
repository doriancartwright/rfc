# Trust Robotics RFC Series

The Trust Robotics RFC Series defines public technical specifications for Physical AI, Humanoid Operating Systems, and TrustBoundary technologies.

RFCs are versioned architectural documents describing interfaces, terminology, runtime behavior, and reference implementations.

## Current RFCs

- TR-RFC-0001 Humanoid Constitution
- TR-RFC-0002 Runtime Objects
- TR-RFC-0003 Humanoid Operating System
- TR-RFC-0004 Body Resource Manager
- TR-RFC-0005 Mission Authority Runtime
- TR-RFC-0006 Humanoid Scheduler
- TR-RFC-0007 Humanoid Memory Manager

- TR-RFC-0009 Humanoid APIs
- TR-RFC-0010 Constitution Compiler
